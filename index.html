jobs.jtplatt.com

A personal distributed system that runs my job search. A daily launchd cron on my Mac crawls four public-sector job boards, scores each posting against a structured inventory of my experience in two tiers (a cheap deterministic filter, then an LLM pass on the survivors), and publishes a static report the server serves behind auth. Rate-limited, robots-aware, and built so an empty scrape never overwrites the last good run.
